## Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK

### Installation

1. Clone the repository:

```sh
git clone https://github.com/Nimastic/QNow-Flutter-IEEEiNTUition-TeamJustKidding.git
```

```sh
cd QNow-Flutter-IEEEiNTUition-TeamJustKidding
```

```sh
flutter pub get
```

```sh
flutter run
```
